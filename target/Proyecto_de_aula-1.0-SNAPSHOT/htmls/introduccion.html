INTRODUCCION A LA APLICACIÓN WEB

Página Web

La página web es parecido a un documento pero la diferencia es que es un documento electrónico escrito en el lenguaje HTML(Hypertext Markup Language). La página web tiene muchas variedades dentro de ella ya que contiene lo que es texto, gráficos, vídeos, animaciones, sonido y elementos interactivos. Las páginas web también contienen una dirección única alo cual se le denomina URL que esta encargada de identificar su ubicación en el servidor, las mayorias de las páginas web generalemente contienen hiperenlaces(son textos e imágenes que hacen referencia a URL de otras paginas web) a otras páginas web.

Sitio web

El sitio web está compuesto de una o más paginas web referida a un asunto común. El sitio web da accesibilidad a otras páginas web a través ya que están enlazadas por una finalidad concreta el sitio web está identificado por una direccion URL única.

Servidor Web

El servidor web o dominio HTTP está encargado de controlar el flujo de datos que entra y sale de un computador conectada a internet. El servidor Web es un programa de aplicación que se encarga de atender las solicitudes HTTP que es realizada por todos los navegadores. Ejemplo:

  • Apache
  • Tomcat

Protocolo HTTP

El protocolo cliente-servidor es el encargado de articular los intercambios de información entre los clientes web y los servidores HTTP. Este protocolo se basa en la operación solicitud/respuesta. Lo cual su funcionamiento es que un cliente establece una conexión con un servidor y envia el mensaje con los datos de la solicitud.

Caracteristicas del protoco HTTP:

  • Cuando hay comunicación entre los cleintes y servidores se realiza a partir de caracteres de 8 bits ya que permite transmitir cualquier tipo de docuemnto.
  • El protocolo HTTP permite que se efectue con excelencia la transeferencia de objetos multimedias.
  • Los verbos básicos que mas se utiliza para que el cliente pueda dialogar con el servidor son GET, POST, HEAD. El GET es el encargado de recoger un objesto, El POST encargado de envir información al servidor y el HEAD es el encargado de solicitar las caracteristicas de un objetos.
  • El HTTP implica una conexión con el servidor en la actualidad ha mejorado mucho este procedimiento ya que permite que la misma conexión se mantenga activa durante un cierto periodo de tiempo, para que haci pueda ser utilizada en posteriores transacciones.
  • Cuando un cliente hace peticiones a un servidor este no influye por la transacciones que se inicieron anteriormente ya que dicho servidor trata las operaciones independientemente.

¿Qué es una aplicación web?

La aplicación web está basada en páginas donde su funcionalidad es en internet, estas páginas web son las que permite que el usuario vea a travéz de algún navegador, estas aplicaciones web estan codificadas en un lenguaje especial para aquello (HTML, JSPs, XML). Una aplicación web también normalmente contiene componentes web como:

  • JavaServer Pages(JSPs)
  • Servlets
  • Java Beans

Las aplicaciones web se almacenan en un servidor, lo cual es un computador que esta encargada de que las aplicaciones web sean lo mas accesible posible. Las aplicaiones web corren en un servidor, para evitar confusiones es importante entender que el software servidor funciona en una computadora servidor. Es muy común que las aplicaiones web hagan uso de una base de datos lo cual esa base de datos está ubicada en la computador-servidor.

Evolución de la Web

La Web ha cambaido mucho desde hace algunos años ahora es mas dinámica existieron por distintos nombres una de las primeras web fue la Web 1.0 que es conocidad por presentar conenido estático. Luego surgió la web 2.o está ya habia evolucionado en distintas areas ya que esta ya permitia vizualizar contenido dinámico o interactivo. La Web 3.0 también fue conocida ya que esta estaria mas ajustada a los intereses de los usuarios. La web 4.0 esta web era considerada como la web emotiva ya que es basada por un ancho de bnda de alta tecnologia ya que permite al usuario obtener un mejor contenido.

La web 1.0 (Estática)

Esta Web fue creada en el 1990 hasta el año 2000, se presentaba de forma de sólo lectura, la cual en esos timepos existian muy pocos. En esta Web los usuarios solo podían visualizar datos estáticos pero no se podía interactuar con la página en esta version de Web no se preocupaban por la seguridad.

Características

  • Páginas Estáticas.
  • Carga Rápida.
  • Diseño sencillo.
  • Poca interaación con el usuario.

La web 2.0 (Web Social)

Esta Web era conocidad como la Web social o interactiva en esta version el usuario ya tenia la facilidad de ingresar datos y poder recibir información, ya en esta web aprecen neuvs tegnologías sin dudarlo en esta web resurgió una nueva era de interactuar con la red en este surgimiento muchas actividades se volcaron a la red, como por ejemplo poder buscar información, desde ese surgimiento las web ganaron mucha importancia a nivel mundial.

Características

  • Dejaron de ser páginas Estáticas.
  • Autenticación de Usuarios.
  • Buscadores.
  • Interaación con el usuario.
  • Navegacion Estructurada.

La web 3.0 (Web Semántica)

La Web 3.0 es una versión de la web y esta web era muy parecidad a la anterior web la version 2.0, en la web 3.0 se puede decir que es una web extendida, con el mayor significado, lo cual brindaba al usuario encontrar respuestas a sus preguntas de forma mas rápida. Las principales tecnologias de la web son: Las que permiten la distribución de contenido, tecnología de internet basada en Ajax, DOM, REST, XML, Blogs.

Características

  • Web Social.
  • Web Semántica.
  • Web 3D.
  • Contiene Inteligencia Artificial.
  • Web centrada en los medios.

La web 4.0 (Internet de las cosas)

Esta Web ofrece un comportamiento mas inteligente que las anteriores web mencionadas, mas predictivo. Esta web es la que abrio la posibilidad de interactuar con el medio físico y que este pueda responder a nuestras necesidades de lo que estemos buscando dentro de la web. La era que estamos hoy en dia se puede considerar como la web 4.0 ya que contine inteligencia artificial, también la nombran como la web simbiótica ya que se puede conectar con personas, procesos, cosas y datos, también creo experiencias mas satisfactorias y nuevas oportiunidades nunca antes vista.

Características

  • Motores de busqueda.
  • Aplicaciones web Inteligentes.
  • Red activa.

Herramientas de desarrollo en el lado de cliente(front-end)

En el ámbito de desarrollos web, el frontEnd se conforma por aquellas tecnologías que corren del lado del cliente, esto quiere decir que todas las tecnologías que se usan del lado del navegador web andan entres lenguajes HTML, CSS y JavaScript. La personas que estan encargadas de FrontEnd debe de conocer las técnicas de experiencia de usuario para brindar una mejor interacción entre la persona y la página de vista, también debe tener conocimiento de diseño de interacción, trabaja con tecnologías ya que es necesario para que pueda consumir datos. En si el frontEnd esta encargado de estilizar las páginas de tal manera que la msima pueda presentar información de forma mas agradable para el usuario que lo esta utilizando.
