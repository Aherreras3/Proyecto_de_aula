¿Qué es HTML?

Existen muchas y diversos tipos de páginas web, y todo lo que se presentan en ellas las distinguen antes las demás, pero a pesar de eso todas las páginas webs tienen algo en común como lo es que todas hechas con HTML.
Para la creación de cualquier tipo de página debe utilizarse HTML sin importar el tipo de contenido que estas vayan a presentar, por eso es importante dominar correctamente HTML antes de pasar a trabajar con otros lenguajes como CSS o JavaScript.
Es importante señalar que las paginas HTML son simples archivos de texto ya que esta es la forma más común de introducir datos a un ordenador. Y cualquier cosa que crees utilizando HTML podrá ser visto en cualquier ordenador independientemente del sistema operativo que utilice.

HiperTexto

Los hipertextos son instrucciones especiales contenidas en HTML que permite que cualquier texto pueda enlazarse con otro contenido de internet. Se les suele llamar "hipervínculos" ("hyperlinks" en inglés) son importantes en la red, sin los hipervínculos esta no podría existir. Ejemplo de hipertexto

HTML 5

HTML 5 es la quinta versión de este lenguaje y la más actualizada hasta la fecha.
Esta versión permite interactuar con datos locales y de servidores de manera más sencilla, proporcionando un mejor soporte multimedia a comparación de sus antecesores.
La principal diferencia entre HTML 5 ante sus antecesores es que esta no depende de APIs ni complementos de terceros que solían dar problemas de compatibilidad, esto hace que esta última versión cargue más rápido y asegure la compatibilidad con todos los navegadores.
HTML 5 tiene bastante nuevas funciones entre las que destacan:

  • Nuevos atributos.
  • Nuevas reglas de nalisis, así como reglas más detalladas.
  • Eliminación de elementos y atributos repetitivos.
  • Adición de página sin conexión(offline).
  • Capacidad para almacenar base de datos MySQL.

Estructura de HTML 5

Los documentos HTML se encuentran estrictamente organizados, cada parte del documento se encuentra diferenciada, declarada y determinada por etiquetas específicas. A continuación, especificaremos cada una de las partes de este documento.

<DOCTYPE>

En primer lugar, se debe indicar el tipo de documento que estamos creando. Esto en HTML se logra con la siguiente etiqueta. 

                
                    <DOCTYPE html>

<html>

La estructura de este leguaje es de tipo arbol y su raiz es el elemento html, este elemento evolvera el resto del codigo. 

                
                    <DOCTYPE html>
                    <html lang=”es”>
                    </html>

El atrubuto lang es el unico elemento necesario al iniciar el documento, este atributo indica el lenguaje humano contenido en el documento que estamos creando.

<head>

El elemento html se encuentra dividido en 2 partes, la cabecera y el cuerpo, el head representa la cabecera y body el cuerpo. 

                
                    <DOCTYPE html>
                    <html lang=”es”>
                    <head>
                    <head>
                    </html>

En este apartado especificamos el titulo de nuestra página web, declaramos set de caracteres incluso podremos incorporar archivos externos de estilo, codigos JavaScript o imagenes.

<body>

Este elemento representa al cuerpo del documento, todo lo que se ecuentre dentro será visible en la página web. 

                
                    <DOCTYPE html>
                    <html lang=”es”>
                    <head>
                    <head>
                        <body>
                        <body>
                    </html>

<meta>

Este elemento se utiliza para definir el juego de caracteres a utilizar para mostrar en el documento, de manera más sencilla esta etiqueta especifica cómo el texto será mostrado en pantalla. Ademas este elemento nos permite agregar otras etiquetas como description o keywords para defifir otros aspectos de la página web. 

                
                    <DOCTYPE html>
                    <html lang=”es”>
                    <head>
                        <meta charset=”iso-8859-1”>
                        <meta><meta name=”description” content=”ejemplo de HTML5” />
                        <meta name=”keywords” content=”HTML5, CSS3, Javascript” />
                    <head>
                    <body>
                    <body>
                    &l/thtml>

<title>

Esta etiqueta simplemente especifica el titulo del documento. 

                
                    <DOCTYPE html>
                    <html lang=”es”>
                    <head>
                        <meta charset=”iso-8859-1”>
                        <meta><meta name=”description” content=”ejemplo de HTML5” />
                        <meta name=”keywords” content=”HTML5, CSS3, Javascript” />
                        <title>Este es el titulo de documento<title/>
                    <head>
                    <body>
                    <body>
                    </html>

<link>

Este elemento va dentro de la cabecera, es usado para incorporar estilos, códigos JavaScript, imágenes o iconos desde archivos externos. Es el más común para incorporar archivos con estilos CSS. 

                
                    <DOCTYPE html>
                    <html lang=”es”>
                    <head>
                        <meta charset=”iso-8859-1”>
                        <meta><meta name=”description” content=”ejemplo de HTML5” />
                        <meta name=”keywords” content=”HTML5, CSS3, Javascript” />
                        <title>Este es el titulo de documento<title/>
                        <link rel=”stylesheet” href=”mystyle.css>
                    <head>
                    <body>
                    <body>
                    </html>

En HTML5 ya no se necesita especificar qué tipo de estilos estamos insertando, por lo que el atributo type fue eliminado. Solo necesitamos dos atributos para incorporar nuestro archivo de estilos: rel y href. El atributo rel significa “relación” y es acerca de la relación entre el documento y el archivo que estamos incorporando por medio de href. En este caso, el atributo rel tiene el valor stylesheet que le dice al navegador que el archivo mystyle.css es un archivo CSS con estilos requeridos para presentar la página en pantalla.
