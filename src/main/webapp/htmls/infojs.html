¿Que es JavaScript?

JavaScript es un lenguaje interpretado, considerado una de las grandes bases en la creación de páginas y por mucho tiempo este solo fue considerado como un complemento para HTML y CSS. La razón por la que empezamos a ver a JavaScript como una herramienta indispensable para la creación de páginas web fue cuando se desarrollaros nuevos motores de interpretación, los cuales fueron creados para acelerar el procesamiento de código. La principal ventaja de estos nuevos motores fue que pasaban el lenguaje JavaScript a lenguaje maquina aumentando el rendimiento, esto permitió superar antiguas limitaciones de rendimiento y ayudo confirmar que JavaScript es la mejor opción para la web.

Importacia de JavaScript

El crecimiento de JavaScript trajo consigo el crecimiento de los navegadores, ahora estos integraban interfaces de programación (APIs) para asistir al lenguaje en funciones elementales. Con esto se busca hacer disponibles nuevas funciones con técnicas de programación sencillas y estándares, expandiendo el alcance del lenguaje y facilitando la creación de programas útiles para la web.

Incorporando JavaScript

Al igual que CSS, existes varias técnicas para incorporar código JavaScript dentro de un documento HTML, pero adelantamos que la más recomendada es la inclusión de archivos externos al igual que ocurre con CSS.
A continuación, pasaremos a explicar dichas técnicas.

En línea

Es una de las técnicas más sencillas, se hace uso de las etiquetas HTMl para dentro de ellas incrustar el código JavaScript, por lo general se suelen usar manejadores de eventos vinculados con el raton como son onclick, onMouseOver entre otros aunque también se suele usar en el teclado. 

                
                    <DOCTYPE html>
                    <html lang=”es”>
                    <head>
                        <title>Título del documento>/title>
                    <head>
                    <body>
                        <p onclick="alert('se realizo un click') "> Mi prueba<p>
                    <body>
                    </html>

El ejemplo anterior mostrara un pequeño recuadro con el mensaje "se realizó un click" cada que se realice dicha acción sobre el texto que contiene el código en su etiqueta.
Al igual que pasó con CSS, esta practica no se recomienda ya que extiende el cogido innecesariamente y puede ser difícil el querer actualizar dichos códigos.

Código embebido

Una mejor forma a comparacion de la anterior es agrupar todos loc códigos en un mismo ligar uilizando la etiqueta <script>, asi podremos utilisar dicho código ulilizando referencias.
El elemento <script> puede ser utilizado en cualquier parte del documento, pero se recomienda colocarlo en la cabecera para tener mayor claridad. 

                
                    <DOCTYPE html>
                    <html lang=”es”>
                    <head>
                    <title>Título del documento>/title>
                        < script >
                         function m_alerta(){
                             alert('realizó clic'); }
                         function e_alerta(){
                             document.getElementsByTagName('p')[0].onclick=m_alerta; }
                        < /script >
                    <head>
                    <div id="texto"&tgt
                        <p> Hacer click<p>
                    </div&tgt
                    </html>

Archivos externos

El código JavaScript crece de una manera exponencial cuando agrgamos funciones y nuevas APIs, por que si utilizamos los codigos embebidos el documento HTML seria enorme. La solucion mas recomendada para evitar este problema es agrupar todo el codigo JavaScript Enun archivo externo y luego añadirlo al documento HTML mediante el atributo src:. 

                
                    <DOCTYPE html>
                    <html lang=”es”>
                    <head>
                        <title>Título del documento>/title>
                        < script src="myjs.js"></script>
                    <head>
                    <body>
                        <p>><p> Mi prueba<p>
                    <body>
                    </html>

Referenciar Elementos

Acualmente existen tes metodod disponibles para referenciar elementos HTML desde un JavaScript, acontinuación, describiremos dichos metodos:

  • getElementsByTagName.

    • Se utiliza para referenciar un elemento por si nombre o palabra clave.

                            
                            <DOCTYPE html>
                            <html lang=”es”>
                            <head>
                            <title>Título del documento>/title>
                                < script >
                                function m_alerta(){
                                     alert('realizó clic'); }
                                function e_alerta(){
                                     document.getElementsByTagName('p')[0].onclick=m_alerta; }
                                < /script >
                            <head>
                            <div id="texto"&tgt
                                <p> Hacer click<p>
                            </div&tgt
                            </html>

  • getElementById.

    • Utilizado para referenciar un elemento especifco, por lo cual no pueden haber mas de un elementos con el mismo id. 

                                
                                <DOCTYPE html>
                                <html lang=”es”>
                                <head>
                                <title>Título del documento>/title>
                                    < script >
                                    function m_alerta(){
                                        alert('realizó clic'); }
                                    function e_alerta(){
                                         document.getElementById('mitexto')[0].onclick=m_alerta; }
                                    < /script >
                                <head>
                                <div id="texto"&tgt
                                    <p id="mitexto"> Hacer click<p>
                                </div&tgt
                                </html>

  • getElementsByClassName.

    • Este elemnto es una nueva incorporacion que nos permite obtener un elemento a partir del valor class

    . 
                            
                            <DOCTYPE html>
                                <html lang=”es”>
                                <head>
                                <title>Título del documento>/title>
                                    < script >
                                    function m_alerta(){
                                        alert('realizó clic'); }
                                    function e_alerta(){
                                         document.getElementsByClassName('mititulo')[0].onclick=m_alerta; }
                                    < /script >
                                <head>
                                <div id="texto"&tgt
                                    <p id="mitexto"> Hacer click/<p>
                                    <h1 class="mititulo"> Hacer click dos<h1>
                                </div&tgt
                                </html>

    En el código anterior se hace referencia a h1 ya que tiene como valor se clase "mititulo".

Nuevos selectores

A comparacion de CSS, JavaScript se queda bastante atras en el temas de los selectores y para ello fue necesario la integracion de nuevos selectores para aumentar el nivel de JavaScript, por eso ahora se puede seleccionar elementos HTML aplicando toda clase de selectores CSS por medio de los métodos querySelector() y querySelectorAll().

  • querySelector.

    • Este método retorna el primer elemento que concuerde con el grupo de selectores especificados entre paréntesis, los selectores osn declarados usando comillas y la misma sintaxis CSS. 

                            
                            <DOCTYPE html>
                                <html lang=”es”>
                                <head>
                                <title>Título del documento>/title>
                                    < script >
                                    function m_alerta(){
                                        alert('realizó clic'); }
                                    function e_alerta(){
                                         document.querySelector("#texto h1:first-child").onclick=m_alerta; }
                                    < /script >
                                <head>
                                <div id="texto"&tgt
                                    <p id="mitexto"> Hacer click/<p>
                                    <h1 class="mititulo"> Hacer click dos<h1>
                                    <h1 class="mititulo"> Hacer click tres<h1>
                                </div&tgt
                                </html>
    El ejemplo anterior retornara el primer hijo (h1) que se ecuentre dento del id texto.

  • querySelectorAll.

    • Este método es muy parecido al anterior, con la diferencia que este de devolverá todos los emelentos que especificastes en el paréntesis en forma de lista. 

                            
                            <DOCTYPE html>
                                <html lang=”es”>
                                <head>
                                <title>Título del documento>/title>
                                    < script >
                                    function m_alerta(){
                                        alert('realizó clic'); }
                                    function e_alerta(){
                                         var lista = document.querySelector("#texto h1");}
                                         for(var i=0; i<lista; i++)
                                            lista[i]..onclick=m_alerta;
                                    < /script >
                                <head>
                                <div id="texto"&tgt
                                    <p id="mitexto"> Hacer click/<p>
                                    <h1 class="mititulo"> Hacer click dos<h1>
                                    <h1 class="mititulo"> Hacer click tres<h1>
                                    <h1 class="mititulo"> Hacer click cuatro<h1>
                                </div&tgt
                                </html>
    El código anterior devolverá todos los elementos h1 que se ecuentren dentro de un id="texto", y con un ciclo(for) se se les puede asiganar eventos a todos esos elementos.
