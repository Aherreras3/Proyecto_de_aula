¿Que es CSS?

CSS es un leguaje de estilos que es complemento de HTML, utilizado para proveer estilos visuales a los elementos del documento permitiendo proveer aspectos como el tamaño, color, fondo, bordes y demás.
Oficialmente CSS no tiene relación con directa con HTML, este fue desarrollado para reducir la complejidad de HTML. Al principio html proveía de estilos esenciales para cada elemento, pero a medida que el lenguaje evolución, la escritura de código se volvió más compleja y HTML por sí mismo no puedo satisfacer las demandas de los diseñadores.
Luego CSS fue adoptado como el leguaje para diseñadores y seguía creciendo en paralelo a HTML, entonces HTML 5 surgió dejando todo el trabajo de diseño a CSS y debido a esto actualmente la integración de HTML y CSS es vital para la creación de cualquier página web.

Conceptos básicos sobre estilos

Al aplicar estilos a un documento HTML se cambia la forma en que se presenta en pantalla. La mayoria de los navegadores proveen estilos por defecto, pero no resultan suficientes para sastifacer las necesidades de los diseñadores.

Tecnicas aplicación CSS

Para cambiar las limitaciones que tienen los navegadores con respecto al estilo, se pueden sobrescribir dichos estilos con diferentes técnicas que describiremos a continuación.

Estilos en linea

Una de las tecnicas mas comunes y usadas es incorporar los estilos CSS en el mismo documento HTML, asignando estilos dentro de las etiquetas utilizando el atributo style. 

                
                    <DOCTYPE html>
                    <html lang=”es”>
                    <head>
                    <head>
                        <p style="font-size: 20px";> Mi prueba<p>
                    </html>

Usar esta técnica es recomendable solo si se desea probar estilos mas no cundo se quiere cambiar todo el documento, la razón es simple ya que tiene que estar colocando estilo a cada uno de los elementos que se desea modificar, incrementando el tamaño del documento a proporciones inaceptables y haciendo más complicado el trabajo de mantener y modificar dicho archivo.

Archivos externos

Una mejor forma a comparacion de la anterio es insertar los etilos en la cabecera del documento y luego usar referencias para afectar a los elementos HTML segun sea necesario. 

                
                    <DOCTYPE html>
                    <html lang=”es”>
                    <head>
                        < style >
                         p{
                            font-size: 20pz;
                             }
                    <head>
                        <p> Mi prueba<p>
                    </html>

Estilos embebidos

Declarar los estilos en la cabecera del documento ahorra espacio y vuelve al código más consistente, pero nos requiere una copia de dichos estilos para cada documento en la que lo queramos aplicar.
La mejor solución es mover todos los estilos a un documento externo y luego utilizar el atributo <link> en cada documento donde se desee utilizar dichos estilos. Este método nos permite cambiar todos los estilos y adaptar los documentos a cada dispositivo de manera más eficaz. 

                
                    <DOCTYPE html>
                    <html lang=”es”>
                    <head>
                        <link rel="stylesheet" href="misestilo.css"/>
                    <head>
                        <p> Mi prueba<p>
                    </html>

Referenciar Elementos

Al almacenar todos nuestros estilos en un archivo externo, nos permite modificar diferentes tipos de elementos, para ello es importantes establecer una referencia de manera correcta.
Existes varios tipos para seleccionar elementos presentes en el documento HTML, los cuales serán afectados por las reglas CSS.

  • Referencia por palabra clave del elemento.
  • Referencia por el atributo id.
  • Referencia por el atributo class.

Referenciando por palabra clave

Al aplicar estilos referenciando por palabra clave del elemento, el cambio se aplica a todos los elementos del mismo tipo, ejemplo, en l siguiente declaracion se modeificaran todos los elementos <p>. 

                
                    p { font-size: 20px }

Con la instruccion anterior le estamos comunicando al navegador que esta regla debe ser aplicada a cada elemento <p> encontrado en el documento HTML, entonces todos los emelentos envueltos en <p> tendran el tamaños de 20 pixeles.

Referenciando con el atributo id

Al referenciar por id nos estamos refiriendo a un elemento único en el documento HTML, por lo tanto, no pueden existir elementos con el mismo id, al referenciar un elemento en el archivo CSS se lo declara con el símbolo # al frente del valor que usaremos para identificar el elemento. 

                
                    <DOCTYPE html>
                    <html lang=”es”>
                    <head>
                        <link rel="stylesheet" href="misestilo.css"/>
                    <head>
                        <p id="text1"> Mi prueba<p>
                    </html>

Realizando ferencia por el atributo id: 

                
                    #text1{
                        font-size: 20px;
                    }

Referenciando con el atributo class

Es recomendable usar el atributo class en lugar de id, ya que este permite ser asignado a varios elementos y los estilos se vayan a aplicar se aplicarán a todos los elementos que pertenezcan a dicha clase.
Para referenciar una clase en CSS debemos declarar un punto antes del nombre de la clase.
Colocando atributo class en HTML: 

                
                    <DOCTYPE html>
                    <html lang=”es”>
                    <head>
                        <link rel="stylesheet" href="misestilo.css"/>
                    <head>
                        <p id="text1"> Mi prueba<p>
                        <p id="text1"> Mi prueba numero 2<p>
                    </html>

Referenciando Class desde CSS: 

                
                    .texto1 { font-size: 20px }

Referenciando con cualquier atributo

Los métodos de referencias estudiados anteriormente cubren la mayor parte de las necesidades, pero a veces no es suficiente.
En la última versión de CSS se han incorporado nuevas formas de referenciar elementos HTML, ahora podemos referenciar un elemento no solo por id y Class sino también a través de cualquier otro atributo. 

                
                    p[name] { font-size: 20px }

La regla anterior cambia los elementos <p> que tienen un atributo llamadon name.
CSS3 perimite combinar "=" con otros para hacer una seleccion mas especifica.
Ejemplo: 

                
                    p[name^=”mi”] { font-size: 20px } 
                    p[name$=”mi”] { font-size: 20px }
                    p[name*=”mi”] { font-size: 20px }

  • La regla con el selector ^= será asignada a todo elemento que contiene un atributo name con un valor comenzado en “mi” (por ejemplo, “mitexto”, “micasa”).
  • La regla con el selector $= será asignada a todo elemento que contiene un atributo name con un valor finalizado en “mi” (por ejemplo “textomi”, “casami”).
  • La regla con el selector *= será asignada a todo elemento que contiene un atributo name con un valor que incluye el texto “mi” (en este caso, el texto podría también encontrarse en el medio, como en “textomicasa”)[1].
En estos ejemplos usamos el elemento <p>, el atributo name, y una cadena de texto al azar como “mi”, pero la misma técnica puede ser utilizada con cualquier atributo y valor que necesitemos.

Referenciando con pseudo clases

CSS3 tambien trajo consigo nuevas pseudo clases que premiten realizar una busqueda de un elemento de una forma aun mas exacta.
Pantilla de prueba: 

                
                    <DOCTYPE html>
                    <html lang=”es”>
                    <head>
                        <title> este es el titulo del documento</title>
                        <link rel="stylesheet" href="misestilo.css"/>
                    <head>
                        <div id="wrapper">
                            <p>><p class="mitexto1"> Mi prueba<p>
                            <p>><p class="mitexto2"> Mi prueba numero 2<p>
                            <p>><p class="mitexto3"> Mi prueba numero 2<p>
                        </div>
                    </html>

Usaremos como primer ejemplo el nth-child. 

                
                    p:nth-child(2){ 
                        background: #999999; 
                       }

La regla anterior localiza al elemento 2 que pertenezca a <p> y que tiene como padre a div, entonces usando pseudo clases podemos aprovechar esta organización y referenciar un elemento especifico sin importar cuanto conocemos sobres sus atributos y el valor de los mismo. 

 
            
                *{ 
                    margin: 0px; 
                   } 
                   p:nth-child(odd){ 
                    background: #999999; 
                   } 
                   p:nth-child(even){ 
                    background: #CCCCCC; 
                   }

La palabra clave odd para la pseudo clase nth-child() afecta los elementos que son hijos de otro elemento y tienen un índice impar. La palabra clave even, por otro lado, afecta a aquellos que tienen un índice par.
Existen otras importantes pseudo clases relacionadas con esta última, como firstchild, last-child y only-child, algunas de ellas recientemente incorporadas. La pseudo clase first-child referencia solo el primer hijo, last-child referencia solo el último hijo, y only-child afecta un elemento siempre y cuando sea el único hijo disponible. Estas pseudo clases en particular no requieren palabras clave o parámetros, y son implementadas como en el siguiente.[1]
ejemplo: 

            
                *{ 
                    margin: 0px; 
                   } 
                   p:last-child{ 
                    background: #999999; 
                   }

Otra importante pseudo clase llamada not() es utilizada realizar una negación: 

            
                :not(p){ 
                    margin: 0px; 
                   }

La utima regla asignara margen 0px a cada elemento del documento excepto los elemento <p>

Nuevos selectores

Hay algunos selectores más que fueron agregados o que ahora son considerados parte de CSS3 y pueden ser útiles para nuestros diseños. Estos selectores usan los símbolos >, + y ~ para especificar la relación entre elementos.[1] 

                
                    div > p.mitexto2{ 
                        color: #990000;

El selector > está indicando que el elemento a ser afectado por la regla es el elemento de la derecha cuando tiene al de la izquierda como su padre. La regla anterior modifica los elementos <p> que son hijos de un elemento <div>.[1] 

                
                    p.mitexto2 + p{ 
                        color: #990000; 
                       }

La regla anterior afecta al elemento <p> que se encuentra ubicado luego de otro elemento <p> identificado con el valor mitexto2 en su atributo class.[1] 

                
                    p.mitexto2 ~ p{ 
                        color: #990000; 
                       }

La regla anterior afecta al tercer y cuarto elemento <p> de la plantilla de ejemplo. El estilo será aplicado a todos los elementos <p> que son hermanos y se encuentran luego del elemento <p> identificado con el valor mitexto2 en su atributo class. No importa si otros elementos se encuentran intercalados, los elementos <p> en la tercera y cuarta posición aún serán afectados.[1]
