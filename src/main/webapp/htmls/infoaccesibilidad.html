Accesibilidad Web

Introducción

La web es el elemento más visible de Internet y está tomando cada vez más importancia, y por esto es considerado uno de los pilares más importantes de la llamada Sociedad Informática. Por ello es importante que en este aspecto no exista ningún tipo de exclusión o discriminación de las personas de esta sociedad, sin importar razones como la edad, conocimientos, idioma, formación, tecnología, cultura, religión, género y por supuesto, discapacidad.

Importacia de la Accesibilidad

Una buena accesibilidad ayuda en gran manera a que un sitio web tenga una mayor aceptación, por esto es importante garantizar la accesibilidad lo más extenso posible sin importar que estas personas tengan alguna deficiencia o no.
Para lograr una buena accesibilidad en la web deben considerarse tres grandes componentes:

  • Contenido

Este es el contenido que muestra la página web y desarrolado por diseñadores web.
Por lo tanto los diseñadores tendran en cuenta cricterios de accesibilidad durante el desarrollo de dicho contenido web.

  • Aplicaciones informaticas

Estas son las aplicaciones usadas por los usuarios para acceder al contenido web, tambien llamados agentes de usuarioo, mas comúnmente llamados navegadores.
Los navegadores deben ofrecer una interfaz accesible y, al mismo tiempo tratar de manera correcta todo el contenido accesible que se visualiza en el el.

  • Programas informaticos

Estas son las aplicaciones que permiten gestionar el contenido web tambien llamadas herramientas de autor.
La herramientas de autor deben ofrecer una interfaz de usuario accesible y, sumultaneamente deben facilitar la creacion de contenido web considerando la accesibilidad, ademas de ayudar a fomentar que las personas con discapacidad se conviertan en creadores de contenido web y no solo meros consumidores.

Necesidad de la Accesibilidad

En los ultimos años la web a crecio de una manera inpensada, ha pasado a ser mas que solo una fuente de informacion gratuita, texto plano que se presenta en pantalla. Actualmente la web tiene muchas funciones que ayudan a la sociedad en si, ejemplo:
-Búsqueda de empleo e interaccion con el trabajo.
-Formación academica.
-Noticias, información, comercio, ocio....
Estas son solo algunas de la razones por lo que una buena accesibilidad es necesaria, ayudando a las personas a acceder a mucho mas contenido ndesde la comodidad de su casa, lugar de trabajo o puesto de trabajo[1].

Pautas de Accesibilidad

La referencia mas usada a nivel mundial es la proporcionada por el Consorcio de la Web o abreviado W3C, a tarves de la Iniciativa para la Accesibilidad a la web abreviado WAI. Este ultimo ha desarrollado diversas guías y documentos para impulsar la creacion de sitios webs accesibles, tanto para los contenidos, como para las herramientas de autor y los agentes de usuario, entre todos estos destacan las Pautas para la Accesibilidad al Contenido Web, abreviadas como WCAG, del ingles (Web Content Accesibility Guidelines)[1].

Pautas contenido web

Se hace referencial a toda la información que se puede encontrar en una pagina o aplicación web, en lo cual se incluye:

  • Información natural como textos, imágenes y sonidos.
  • Código o marcado que define la estructura, presentación, etc.

Algunas de recomendaciones o pautas que se recomiendan aplicar al contenido web son las siguientes:

Perceptible

  • Proporcione alternativas textuales para contenido no textual.
  • Proporciones subtítulos y otras alternativas para multimedia.
  • Cree contenido que se pueda presentar de diferentes formas.
  • Facilite que los usuarios puedan ver y oír el contenido[2].

Operable

  • Proporcione acceso a todas las funcionalidades mediante teclado.
  • Proporcione a los usuarios tiempo suficiente para leer y utilizar el contenido.
  • Evitar usar contenido que pueda causar convulsiones o reacciones físicas.
  • Facilite a los usuario la manera de navegar y encontrar contenido[2].

Comprensible

  • Proporcione texto legible y comprensible.
  • Proporcione contenido que sea predecible en apariencia y operación.
  • Ayude a los usuarios a evitar y corregir errores.
  • Facilite a los usuario la manera de navegar y encontrar contenido[2].

Pautas herramientas de creación de contenido

Estas herramientas on la que los "autores" (desarrolladores web, diseñadores, redactores, etc utilizan para producir contenido web.

  • Que las herramientas sean accesibles para las personas con discapacidad.
  • Ayudar a los autores a crear contenido web más accesible[2].
